# Juridisch Kader (Nieuw Strafwetboek)

## Strafniveaus (Vanaf 8 april 2026)
- **Niveau 4 (5-10 jaar):** Deelname aan terroristische groep.
- **Niveau 6 (15-20 jaar):** Leiden van een terroristische groep.
- **Niveau 3 (Nieuw):** Verheerlijken van terrorisme (zelfstandig misdrijf).

## Definities
- **Terroristische Groep:** Gestructureerde vereniging (>2 personen) die terroristische misdrijven pleegt.
- **Verzwarend:** Terroristische drijfveer levert +1 strafniveau op bij elk basisdelict.
